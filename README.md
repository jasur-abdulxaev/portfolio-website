# Jasurbek Abdulxayev — Software Developer & Educator Portfolio

![Portfolio Preview](hero_bg.png)

Welcome to the repository for my personal portfolio website. This project serves as a digital resume and a demonstration of my work, specifically focusing on backend development, software architecture, and technology education.

## 🌍 Live Website

**[Visit the Live Portfolio](https://portfolio-website-ten-iota-15.vercel.app/)**

## ✨ Features

- **Modern & Responsive UI**: Clean, dark-mode focused aesthetic built entirely with Vanilla CSS.
- **Dynamic Animations**: Includes scroll-reveal effects, dynamic skill bar animations, and a custom particle background via JavaScript.
- **Interactive Contact Form**: A fully functional contact form integrated directly with **Supabase**, allowing users to send messages that are securely stored in a cloud database.
- **Project Showcase**: Detailed cards emphasizing system design and backend architecture for various deployed projects.
- **Performance Optimized**: Zero heavy frontend frameworks used—just pure HTML, CSS, and JS ensuring incredibly fast load times.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Variables, Grid, Flexbox), Vanilla JavaScript (ES6+).
- **Backend/Database**: [Supabase](https://supabase.com/) (PostgreSQL) for capturing and storing contact form submissions.
- **Deployment**: [Vercel](https://vercel.com/) for fast, continuous delivery.

## 📂 Project Structure

- `index.html` — The main structure and content of the one-page portfolio.
- `style.css` — All styles, including CSS variables directly managing the custom color scheme and typography.
- `main.js` — Client-side logic for intersection observers, animations, mobile navigation, and Supabase database interactions.

## 🚀 Local Deployment

If you'd like to run this project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/jasur-abdulxaev/portfolio-website.git
   ```
2. Open the project folder.
3. Simply double-click `index.html` or run it through a local HTTP Server like VS Code's "Live Server" extension.
